# calculator

## Existing Operations
The calculator currently supports the following operations:
*   `+`: Addition
*   `-`: Subtraction
*   `*`: Multiplication
*   `/`: Division
*   `^`: Exponentiation (e.g., `2 ^ 3` for 2 cubed)

## Possible Future Operations
Here is a list of common calculator operations that could be added:
*   Square root (`sqrt`)
*   Modulo (`%`)
*   Logarithm (`log`)
*   Trigonometric functions (`sin`, `cos`, `tan`)
*   Factorial (`!`)
